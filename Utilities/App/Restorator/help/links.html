Images

The Paint program shipping with Windows has very limited functionality. We recommend 3rd party imaging programs.
  • SnideSoft's SnIco Edit: freeware to create nice looking icons
  • Aha-soft's IconXP: shareware icon editor
  • Paint Shop Pro from JASC Software offers professional features for a low price.
  • Image Resizer is our own software that facilitates batch resizing, scaling, cropping, and rotation of images
