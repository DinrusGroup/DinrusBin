*UDS
* The first line of the file must contain "*UDS".
*
* The format of the entries is defined as:
*
*    StructureName,MemberName,MemberType,MemberCount,MemberComment
*
*     where MemberType is one of the following:
*         B = BYTE
*         W = WORD
*         D = DWORD
*         A = Pointer to ANSI Buffer
*         U = Pointer to UNICODE Buffer
*         C = array of characters (null-terminated)
*         W = array of wide-characters (null-terminated)
*     and MemberCount is numeric.
* There is no automatic alignment all bytes must be defined.
*
* MemberComment is always ignored.  If MemberCount is not specified,
* then the default is 1.  If MemberType is not specified, then the
* default is D (DWORD).  MemberType, MemberCount, and MemberComment
* are optional, but in order to specify a MemberCount, you must
* assign a MemberType.
*
